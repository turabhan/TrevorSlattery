Описание проекта:

Это backend-приложение для фруктового магазина, разработанное на Java с использованием Spring Boot. Проект предоставляет REST API для просмотра ассортимента фруктов и оформления заказов. Пользователи могут получить список доступных фруктов с ценами, узнать количество товара на складе, а также сделать заказ, указав, какие фрукты и в каком количестве они хотят приобрести.

Функциональные возможности:

Просмотр всего ассортимента фруктов, Получение информации о конкретном фрукте по ID, Добавление новых фруктов в ассортимент, Обновление информации о фрукте (цена, количество), Удаление фрукта из ассортимента, Создание заказа с проверкой наличия товара, Просмотр истории заказов
