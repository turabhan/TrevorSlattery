# 🍎 Fruit Store Backend

## 📌 Описание проекта

Backend-приложение интернет-магазина фруктов, реализованное на Java с использованием Spring Boot.  
Предоставляет REST API для работы с товарами, корзиной и заказами.

Архитектура приложения построена по классической схеме:

Controller → Service → Repository → Database


## ⚙️ Функциональные возможности

- Просмотр списка товаров
- Получение товара по ID
- Добавление и удаление товаров
- Работа с пользователями
- Добавление и удаление товаров из корзины
- Просмотр корзины
- Создание заказов
- Просмотр заказов


## 🧱 Технологический стек

- Java 25
- Spring Boot
- Spring Data JPA
- Hibernate
- Lombok
- PostgreSQL
- Maven / Gradle


## 🧪 Тестирование

API протестировано с помощью Postman.  
Основные CRUD-операции работают корректно.


## 📈 Дальнейшее развитие

- Валидация входных данных
- Реализация оформления заказа из корзины
- Глобальная обработка ошибок
